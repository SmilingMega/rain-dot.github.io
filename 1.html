基本概念

概念 描述
随机试验 具有以下特点的试验称为随机试验，通常用字母 EEE 表示：
\qquad 1. 可以在相同条件下重复进行；
\qquad 2. 每次试验的结果不止一个，并且事先能明确所有可能的结果；
\qquad 3. 进行一次试验之前不能确定哪一个结果会出现。
样本空间 Ω\OmegaΩ 和样本点 ω\boldsymbol{\omega}ω 样本空间 Ω\OmegaΩ：随机试验所有可能的结果组成的集合称为样本空间。
样本点 ω\boldsymbol{\omega}ω：样本空间的元素，即随机试验的每一个可能的结果称为样本点。
随机事件 随机事件：样本空间 Ω\OmegaΩ 的的子集称为随机事件，简称为事件，通常用 A,B,CA,B,CA,B,C 表示。
事件发生：在每次试验中，当且仅当这一子集中的一个样本点出现时，称这一事件发生。
分类：
\qquad (1) 基本事件：由一个样本点组成的单点集。
\qquad (2) 复合事件：由至少两个基本事件组成。
\qquad (3) 必然事件：样本空间 Ω\boldsymbol{\Omega}Ω 包含所有的样本点，它是 Ω\boldsymbol{\Omega}Ω 自身的子集，在每次试验中它总是发生的，称为必然事件。记为 Ω\boldsymbol{\Omega}Ω 。
\qquad (4) 不可能事件：空集 ∅\emptyset∅ 不包含任何样本点，它也作为样本空间的子集，在每次试验中都不发生，称为不可能事件，记为 ∅\emptyset∅。
事件的关系 （1）包含关系：A⊂B⇔A \subset B \LeftrightarrowA⊂B⇔ 事件 AAA 发生一定导致事件 BBB 发生。
（2）事件相等：A⊂B,且B⊂AA \subset B, \text{且} B \subset AA⊂B,且B⊂A，则事件 A=BA = BA=B
（3）AAA 和 BBB 的和事件：记为：A⋃BA \bigcup BA⋃B（或 A+BA+BA+B）⇔\Leftrightarrow⇔ A,BA, BA,B 至少有一个发生时，事件 A⋃BA \bigcup BA⋃B 发生。类似，称 ⋃k=1nAk\bigcup_{k=1}^nA_k⋃k=1n​Ak​ 为 nnn 个事件 A1,A2,⋯ ,AnA_1,A_2,\cdots,A_nA1​,A2​,⋯,An​ 的和事件。
（4）AAA 和 BBB 的积事件：记为：A⋂BA \bigcap BA⋂B（或 ABABAB）⇔\Leftrightarrow⇔ A,BA, BA,B 同时发生时，事件 A⋂BA \bigcap BA⋂B 发生。类似，称 ⋂k=1nAk\bigcap_{k=1}^nA_k⋂k=1n​Ak​ 为 nnn 个事件 A1,A2,⋯ ,AnA_1,A_2,\cdots,A_nA1​,A2​,⋯,An​ 的积事件。
（5）AAA 和 BBB 的差事件：事件 A−BA-BA−B ⇔\Leftrightarrow⇔ AAA 发生且 BBB 不发生时的事件，也记为 AB‾A\overline{B}AB。注：A−B=A−AB=AB‾A-B = A-AB = A\overline{B}A−B=A−AB=AB
（6）互斥（互不相容）事件：AB=∅AB = \emptysetAB=∅ ⇔\Leftrightarrow⇔ A,BA,BA,B 不能同时发生。
（7）对立（互逆）事件：A⋃B=ΩA \bigcup B = \OmegaA⋃B=Ω 且 A⋂B=∅A \bigcap B = \emptysetA⋂B=∅ ⇔\Leftrightarrow⇔ A,BA,BA,B 在一次试验中必然发生且只能发生一个。AAA 的对立事件记为 A‾\overline{A}A
事件的运算关系 （1）交换律：A⋃B=B⋃A;A⋂B=B⋂BA \bigcup B = B \bigcup A; A \bigcap B = B \bigcap BA⋃B=B⋃A;A⋂B=B⋂B
（2）结合律：A⋃(B⋃C)=(A⋃B)⋃C;A⋂(B⋂C)=(A⋂B)⋂CA \bigcup (B \bigcup C) = (A \bigcup B) \bigcup C; A \bigcap (B \bigcap C) = (A \bigcap B) \bigcap CA⋃(B⋃C)=(A⋃B)⋃C;A⋂(B⋂C)=(A⋂B)⋂C
（3）分配率：A⋃(B⋂C)=(A⋃B)⋂(A⋃C)A\bigcup (B \bigcap C) = (A \bigcup B) \bigcap (A \bigcup C)A⋃(B⋂C)=(A⋃B)⋂(A⋃C)
（4）德摩根律（对偶率）：A⋃B‾=A‾⋂B‾,A⋂B‾=A‾⋃B‾\overline{A \bigcup B} = \overline{A} \bigcap \overline{B}, \overline{A \bigcap B} = \overline{A} \bigcup \overline{B}A⋃B​=A⋂B,A⋂B​=A⋃B

事件的独立性

两个事件的独立性

  • 定义
    • 设 A,BA,BA,B 是两个事件，如果满足等式 P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)，则称事件 A,BA,BA,B 相互独立，简称事件 A,BA,BA,B 独立
  • 独立性的等价说法
    • 若 0<P(A)<10 < P(A) < 10<P(A)<1，则事件 A,BA,BA,B 独立 ⇔\Leftrightarrow⇔ P(B)=P(B∣A)⇔P(B)=P(B∣A‾)⇔P(B∣A)=P(B∣A‾)P(B) = P(B|A) \Leftrightarrow P(B) = P(B|\overline{A}) \Leftrightarrow P(B|A) = P(B|\overline{A})P(B)=P(B∣A)⇔P(B)=P(B∣A)⇔P(B∣A)=P(B∣A)
  • 独立性的性质
    • 若事件 A,BA,BA,B 相互独立，则 AAA 与 B‾\overline{B}B，A‾\overline{A}A 与 BBB，A‾\overline{A}A 与 B‾\overline{B}B 也相互独立。
  • 三个事件的独立性
    • 定义
      • 设 A,B,CA,B,CA,B,C 是三个事件，如果满足等式：
        • P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)
        • P(AC)=P(A)P(C)P(AC) = P(A)P(C)P(AC)=P(A)P(C)
        • P(BC)=P(B)P(C)P(BC) = P(B)P(C)P(BC)=P(B)P(C)
        • P(ABC)=P(A)P(B)P(C)P(ABC) = P(A)P(B)P(C)P(ABC)=P(A)P(B)P(C)
      • 则称三个事件 A,B,CA,B,CA,B,C 相互独立。
      • 若仅满足等式（1）（2）（3），则称三个事件 A,B,CA,B,CA,B,C 两两独立。
    • 性质
      • 若三个事件 A,B,CA,B,CA,B,C 相互独立，则任意两个事件的和、积、差构成的新事件与另外一个事件或者它的逆事件是相互独立的。例如事件 A⋃BA \bigcup BA⋃B 与事件 C‾\overline{C}C 相互独立。
